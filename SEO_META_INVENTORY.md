Alla sidor har canonical URLs implementerade via `CanonicalUrl`-komponenten:

- 1 FAQ-sida med explicit `<CanonicalUrl path="/faq" />`

### ✅ OG/Twitter Meta
Alla sidor genererar automatiskt OG och Twitter meta tags via `AdvancedSEO`-komponenten som inkluderar:
- `og:title` (baserat på seoTitle)
- `og:description` (baserat på seoDescription)
- `og:url` (baserat på canonicalPath)
- `og:type` ("article" för guides, "website" för FAQ)
- `og:image` (standard: og-image.png)
- `twitter:card` ("summary_large_image")
- `twitter:title`
- `twitter:description`
- `twitter:image`

### ✅ JSON-LD Schema
- **Guides:** Article schema med author, publishedTime, section, tags
- **FAQ:** FAQPage schema med alla 32 frågor och svar

## Implementeringsdetaljer

### GuideLayout Component
Alla 8 guides använder `GuideLayout`-komponenten som automatiskt hanterar:
- AdvancedSEO med type="article"
- CanonicalUrl (om canonicalPath anges)
- Breadcrumbs med strukturerad data
- Article metadata (author, publishedTime, section, tags)

### AdvancedSEO Component
`AdvancedSEO`-komponenten genererar automatiskt:
- Standard HTML meta tags (title, description, keywords)
- Open Graph meta tags för social sharing
- Twitter Card meta tags
- JSON-LD structured data (Organization, Breadcrumb, Article, FAQ)

### Inga visuella ändringar
All SEO-metadata hanteras i `<head>` via react-helmet-async. Inga ändringar har gjorts i sidornas visuella layout eller innehåll.
